# 개발 로드맵

## 0.1.0 — 현재 구현

- 기존 JavaScript 계산 로직의 TypeScript 도메인 엔진화
- 세율·공제 규칙 JSON 분리
- 단독명의 계산
- 공동명의 계산을 UI에서 엔진으로 이동
- 전체 부동산 양도가액 기준 고가주택 판정
- 입력 검증과 미지원 사건 차단
- MCP stdio 서버
- `validate`, `calculate`, `list_supported` 도구
- 기존 계산기 회귀 테스트와 경계 테스트

## 0.2.0 — 다음 단계

- 법률·시행령·시행규칙별 규칙 검증표 작성
- 동일 연도 기본공제 기사용 로직의 사례 확장
- 필요경비 종류별 적격성·증빙 상태 검증
- 계산 결과와 국세청 신고서 필드 매핑
- 계산 내역 감사 로그
- HTML 화면이 새 TypeScript 엔진을 사용하도록 교체

## 0.3.0

- 국가법령정보 공동활용 API 기반 근거 조회
- 법령 변경 감지 리포트
- 법령 근거 원문과 계산 규칙 연결
- MCP Inspector 기반 호출 테스트 자동화

## 0.4.0

- 매도·취득 계약서 이미지 필드 추출용 별도 MCP
- 추출값별 신뢰도와 원문 위치 반환
- 사용자 확인 화면
- 누락 증빙 체크리스트

## 1.0.0 목표

- 계약서·증빙 분석
- 세액 계산
- 신고서 작성용 JSON 생성
- 세무사 검토·승인 흐름
- 공식 신고서 초안 출력
