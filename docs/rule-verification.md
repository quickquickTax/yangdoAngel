# 세법 규칙 검증 절차

현재 `2026-04-21.json`은 기존 HTML 계산기의 규칙을 분리한 초기 자료이며 검증 상태는 `pending_professional_review`입니다.

실무 배포 전 규칙별로 아래 절차를 수행합니다.

1. 법률 본문 확인
2. 위임 시행령 확인
3. 위임 시행규칙 확인
4. 별표·서식 확인
5. 관련 조세특례제한법 및 지방세법 확인
6. 시행일과 양도일 적용관계 확인
7. 국세청 신고서식 및 작성요령 대조
8. 경계값 테스트 추가
9. 검토자·검토일·근거 URL 기록
10. 규칙 파일을 새 버전으로 고정

법령 변경을 감지하더라도 계산 규칙을 자동 배포하지 않습니다. 사람이 계산 영향과 경과규정을 확인한 뒤 테스트를 통과한 규칙만 배포합니다.
